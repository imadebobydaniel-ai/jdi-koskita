# 1. Pendahuluan

Administrator Manual merupakan panduan resmi penggunaan **Portal Admin JDI KOSKITA**. Dokumen ini menjelaskan fungsi setiap menu, proses operasional, serta hak akses pengguna agar pengelolaan rumah kos dapat dilakukan secara efektif, aman, dan terstruktur.

---

# 2. Hak Akses Pengguna

Sistem menggunakan **Role Based Access Control (RBAC)**.

| Role                | Hak Akses                                 |
| ------------------- | ----------------------------------------- |
| Super Administrator | Akses penuh seluruh sistem                |
| Owner Kos           | Mengelola seluruh operasional dan laporan |
| Manager Operasional | Mengelola aktivitas operasional           |
| Admin Kos           | Mengelola data harian                     |
| Staff Keuangan      | Tagihan, pembayaran, laporan              |
| Teknisi             | Maintenance                               |
| Cleaning Service    | Status kebersihan kamar                   |

---

# 3. Login Administrator

Langkah:

1. Buka Portal Admin.
2. Masukkan Email.
3. Masukkan Password.
4. Klik **Masuk**.
5. Sistem menampilkan Dashboard sesuai hak akses.

# 22. Troubleshooting

### Tidak dapat login

* Periksa Email dan Password.
* Pastikan akun masih aktif.
* Gunakan fitur Reset Password.

### Data tidak muncul

* Periksa koneksi internet.
* Muat ulang halaman.
* Hubungi Super Administrator jika masalah berlanjut.

### Upload gagal

* Pastikan ukuran dan format file sesuai ketentuan.
* Coba unggah kembali.

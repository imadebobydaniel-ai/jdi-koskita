# 1. Pendahuluan

---

# 2. Versioning Standard

JDI KOSKITA menggunakan format **Semantic Versioning (SemVer)**:

```text
MAJOR.MINOR.PATCH
```

Contoh:

```text
1.0.0
1.1.0
1.2.5
2.0.0
```

Keterangan:

* **MAJOR** → Perubahan besar atau fitur utama.
* **MINOR** → Penambahan fitur tanpa mengubah kompatibilitas.
* **PATCH** → Perbaikan bug atau peningkatan kecil.

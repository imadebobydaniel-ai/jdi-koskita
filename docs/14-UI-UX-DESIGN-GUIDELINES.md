# 1. Tujuan

Dokumen ini menjadi pedoman desain antarmuka (UI) dan pengalaman pengguna (UX) untuk seluruh modul JDI KOSKITA agar memiliki tampilan yang konsisten, profesional, modern, dan mudah digunakan.

---

# 2. Filosofi Desain

Prinsip desain yang digunakan:

* Clean Design
* Modern Interface
* User Friendly
* Responsive
* Mobile First
* Fast Navigation
* Accessibility
* Consistency

---

# 3. Identitas Visual

## Logo

Gunakan logo resmi **Jaya Digital Integrate (JDI)** pada:

* Website
* Login
* Dashboard
* Android Splash Screen
* PDF Invoice

```
Top Navbar

↓

Sidebar

↓

↓
```

---

# 8. Komponen Standar

Gunakan komponen Bootstrap 5:

* Navbar
* Card
* Table
* Modal
* Toast
* Alert
* Badge
* Dropdown
* Pagination
* Accordion
* Tabs
* Breadcrumb
* Offcanvas
* Form Controls

---

# 9. Form Design

Setiap form wajib memiliki:

* Label
* Placeholder
* Validasi
* Pesan Error
* Tombol Simpan
* Tombol Batal

---

# 10. Dashboard

Dashboard harus menampilkan:

* Total Gedung
* Total Kamar
* Kamar Terisi
* Kamar Kosong
* Pendapatan Bulanan
* Tagihan Belum Dibayar
* Pengaduan Aktif
* Grafik Pendapatan

---

# 11. CMS

CMS harus mendukung:

* Drag & Drop Gambar
* Upload Banner
* Upload Galeri
* Atur Harga
* Atur Diskon
* Atur Status Kamar
* Atur Fasilitas
* Preview Sebelum Publish

---

# 12. Responsive Design

Breakpoint:

* Mobile (<576 px)
* Tablet (576–991 px)
* Laptop (992–1199 px)
* Desktop (≥1200 px)

Semua halaman wajib responsif.

---

# 13. Animasi

Gunakan animasi ringan:

* Fade In
* Slide Up
* Hover Effect
* Loading Spinner

Hindari animasi berlebihan.

---

# 14. Accessibility

* Kontras warna yang baik.
* Navigasi menggunakan keyboard.
* Alt text pada gambar.
* Ukuran tombol mudah disentuh pada perangkat mobile.

---

# 15. UX Guidelines

* Maksimal 3 klik menuju fitur utama.
* Tampilkan loading saat proses berlangsung.
* Berikan notifikasi sukses atau gagal.
* Gunakan bahasa yang mudah dipahami pengguna.

---

# 16. Ringkasan

Standar UI/UX ini memastikan seluruh modul JDI KOSKITA memiliki tampilan yang seragam, profesional, mudah digunakan, dan memberikan pengalaman pengguna yang baik di Website maupun Android.
